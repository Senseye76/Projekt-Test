# Riskanalys


## Identiferade riskområden 

### Inloggning 
- Risk för att felmedelanden inte visas rätt
- Risk för att obehriga får åtkoms till din kundvagn
- Risk för att användaren kan logga in utan giltiga uppgifter
- Risk för att inloggningssession inte avslutas korrekt vid utloggning

### Datahantering
- Risk för dataförlust vid upptatering
- Risk för dataförlust vid borttagning
- Risk för att data inte visas på korrekt sätt

### Behörighet
- Risk för att användare kan manipulera URL för att nå otillåtna sidor


### Navigering
- Risk för brutna länkar mellan de olika sidorna
- Risk för inkosekvent beteended vid sidbyte


### Anvädarupplevelse 
- Risk för att odydliga meddelanden
- Risk för att användaren inte förstår nästa steg
